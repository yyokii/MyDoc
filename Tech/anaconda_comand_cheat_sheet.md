| 用途                                           | コマンド                                            | メモ |
| ---------------------------------------------- | --------------------------------------------------- | ---- |
| 環境作成                                       | `conda create -n [name] python=[version] [library]` |      |
| 環境一覧                                       | `conda info -e`                                     |      |
| 環境有効化                                     | `activate [name]`                                   |      |
| 環境無効化                                     | `deactivate [name]`                                 |      |
| 環境削除                                       | `conda remove -n [name] --all`                      |      |
| パッケージインストール<br />（バージョン指定） | `conda install [name]==x.x.x`                       |      |
